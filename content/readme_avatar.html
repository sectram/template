Avatar based course: Data Classification

Change and Upload Logo:

1. Click on Content Template

2. File and choose index.html

3. Editor Type and choose Visual Editor

4. Click on Upload File or Image

5. Click on Browse Server

6. Double Click on img folder

7. Click on Upload or Drag the image on Upload

8. Edit the file name on index.html "logo.png" and replace the name of the image you want to put on a logo

9. Click "Save"

Change and Upload Logo on Certificate:

1. Click on Content Template

2. File and choose index.html

3. Editor Type and choose Visual Editor

4. Click on Upload File or Image

5. Click on Browse Server

6. Double Click on img folder

7. Click on Upload or Drag the image on Upload

8. Edit the file name on 0280.html "logo-grey.png" this is the certificate top logo and replace the name of the image you want to put on a logo certificate

8. Edit the file name on 0280.html "logo-grey.png" this is the certificate bottom logo and replace the name of the image you want to put on a logo certificate

9. Click "Save"

Delete a Chapter on Template

1. Click on Content Template

2. File and choose course.js

3. Editor Type and choose Code Mirror

4. Choose what Chapter do you want to delete and leave a Comment "//" from the cards or lesson you want to delete it

5. Click "Save"

Edit the Exam styles (colors, fonts)

1 - EXAM title

The styles of this header are the same as the headers of all other pages. If you change these styles, they will change for the headers of all pages.
These styles are:

        /* static/css/style.css - line 573 */
        .lesson .lesson-body h1 {
        font-size: 25px;
        line-height: 34px;
        padding-bottom: 7px;
        padding-top: 7px;
        font-weight: bold;
        }

2 - Image

You can replace the image that is in the folder static/exam/fin-exam.png.
You can also change the name of the image file in "0164.html":

        <div class="col-md-12">

        <img src="%static%/exam/fin-exam.png">

Removing avatar from the Template:

1. Click on Content Template

2. Look and select 0020.html in the dropdown menu and find the lesson30 class.

3. Look for carousel class under the lesson30 div class.

4. Comment the avatar slideItem class that you want to remove in the template.

5. Look and select Lesson10.js in the dropdown menun and find Lesson30 function.

6. Inside the lesson30 function comment the all condition of the avatar that need to remove from the template.
